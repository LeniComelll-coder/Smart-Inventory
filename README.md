# Smart-Inventory 📦

**Smart-Inventory** adalah proyek desain antarmuka (UI) sederhana untuk sistem manajemen inventaris barang. Proyek ini dibuat untuk mensimulasikan alur penggunaan aplikasi mulai dari autentikasi hingga pengelolaan data di dashboard.

## 🚀 Tentang Proyek
Proyek ini merupakan hasil eksperimen saya dalam merancang struktur halaman web menggunakan HTML. Fokus utamanya adalah menciptakan alur pengguna yang logis untuk sebuah sistem inventaris.

### Halaman yang Tersedia:
* **Login & Registrasi:** Halaman awal untuk akses pengguna ke dalam sistem.
* **Dashboard:** Halaman utama yang menampilkan ringkasan data inventaris.
* **Halaman Pembeli:** Antarmuka khusus untuk mengelola atau melihat data pembeli/pelanggan.

### Tech Stack:
* **Bahasa:** HTML5
* **Konsep:** UI/UX Design & Frontend Development Dasar

## 🛠️ Cara Melihat Proyek
Karena proyek ini berbasis HTML statis, kamu tidak perlu menginstal apa pun. Cukup ikuti langkah ini:

1.  **Clone repositori:**
    ```bash
    git clone [https://github.com/LeniComelll-coder/Smart-Inventory.git](https://github.com/LeniComelll-coder/Smart-Inventory.git)
    ```
2.  Buka folder proyek dan klik dua kali pada file `Login.HTML` untuk memulai simulasi dari halaman awal di browser kamu.
